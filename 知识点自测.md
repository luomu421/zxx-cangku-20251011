- Q2：补全 `style` 属性，使`<div style=""></div>`的宽为 `100px`、高为 `100px`。
  A2: `<div style="width:100px; height:100px"></div>`

## JavaScript

- Q1：JS 中有哪些**基本数据类型**？
  A1：数字类型，字符串类型，布尔类型，未定义类型，对象类型，复合类型  

- Q2：如何通过 dom 操作获取 `id` 为 `test` 的元素，为其添加一个 "click" 事件？
  A2：let text = document.getElementById('text');
      text.onclick = function(){
        console.log('元素被点击了');
      }

- Q3：箭头函数对比普通函数的区别有哪些？
  A3：1.this指向不同，普通函数谁调用指向谁，箭头函数this好是父级作用域的
      2.当形参只有一个时，可以省略（）
      3.只有返回值或只有一句代码的时候，{}可以省略
      4.没有arguments

- Q4：已知一个变量 `num === 1`，写一个模版字符串，使其显示结果为 `“class1”`，即在 `"class"` 后面插入变量 `num`。
  A4：`<div>{}</div>`
       let str = `class${num}`;
       let box = document.querySelector('div');
       box.innerHTML = str;
